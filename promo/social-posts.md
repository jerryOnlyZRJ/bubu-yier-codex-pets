### Body

```text
我把布布和一二做成了 Codex Desktop 的自定义桌面宠物。

这次整理成了一个可以直接安装的 GitHub 项目：

1. 包含 bubu 和 yier 两个宠物
2. 每个宠物都有独立的 pet.json 和 spritesheet.webp
3. 仓库里放了预览图
4. 支持一行脚本安装到 ~/.codex/pets/

安装方式也很简单：

git clone https://github.com/jerryOnlyZRJ/bubu-yier-codex-pets.git
cd bubu-yier-codex-pets
./scripts/install.sh

安装后重启 Codex Desktop，或者重新打开宠物选择器，就可以切换到 bubu 或 yier。

项目地址：
https://github.com/jerryOnlyZRJ/bubu-yier-codex-pets

如果你也在折腾 Codex Desktop 的自定义宠物，可以直接拿这个仓库参考目录结构和安装方式。
```
